Substantial Functional Feature Description

The substantial feature chosen by the group is the ability to upvote or downvote a post. Initial discussion revealed that one noticable feature that other software development forums offer is the up/down vote service. This subtle feature acts as a key component to the way in which users interact with the forums. Posts with higher up-votes tend to become more popular at a faster rate than its counterparts which this enables good contributions to be found quicker and easier.

The agreed upon functional limit for the prototype was the feature attached to posts. However, that is not to say that the functionality could not be stretched to other features of the system. Extension ideas included voting on topics and even comments. The full implementation of this feature would not hold back on how it could be utilized by the systems users.

The requirements for the function feature can be found above (1.5).
